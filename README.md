# Rivora Final Premium Ecommerce Frontend

Premium React + Vite + Tailwind CSS + Framer Motion ecommerce frontend.

## Run

```bash
npm config set registry https://registry.npmjs.org/
npm install
npm run dev
```

Open: http://localhost:5173/

## Features

- Clickable navbar links and routes
- Click-based premium Shop Now mega menu
- Product details pages
- Search overlay with clickable results
- Wishlist page
- Cart drawer with quantity controls
- Checkout page
- Responsive design for phone, tablet, laptop
- Framer Motion animations
- Logo image in navbar
- Hero background image used in main section
